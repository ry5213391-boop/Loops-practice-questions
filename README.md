## 💡 Key Concept: Digit Extraction

The foundation of today's learning was understanding how to extract and process each digit of a number.

```python
while n != 0:
    digit = n % 10      # Extract the last digit
    # Process the digit
    n //= 10            # Remove the last digit
```

This simple pattern can solve many number-based programming problems.

## 🧠 Problem-Solving Pattern

Almost every number manipulation problem follows this structure:

```python
while n != 0:
    digit = n % 10

    # Perform required operation

    n //= 10
```

The only thing that changes is the operation performed on each extracted digit.
